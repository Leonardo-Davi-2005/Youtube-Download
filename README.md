# 🎬 YouTube Downloader

Um aplicativo desktop desenvolvido em **Python** para baixar vídeos e músicas do YouTube de forma simples e rápida.

## ✨ Recursos

- 📹 Download de vídeos em MP4
- 🎵 Download de músicas em MP3
- 🎚️ Seleção da qualidade disponível
- 📊 Barra de progresso em tempo real
- ⚡ Exibição da velocidade de download
- ⏳ Tempo restante estimado
- 📁 Escolha da pasta onde salvar o arquivo
- 🖥️ Interface gráfica moderna desenvolvida com Tkinter
- 🪟 Executável para Windows (.exe)

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/YouTubeDownloader.git
```

Entre na pasta:

```bash
cd YouTubeDownloader
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

---

## ▶️ Executando

```bash
python Youtube_Downloader.py
```

---

## 🛠️ Gerando o Executável

```bash
pyinstaller --onefile --windowed --icon=icone.ico --name "YouTube Downloader" Youtube_Downloader.py
```
